# Waterbath Project - IMDEA RobotLab

Developed at IMDEA Materials Institute.

## Project Overview

This repository is a mixed collection of CAD files, 3D-printable components, and supporting resources developed for the Waterbath project at the IMDEA Materials RobotLab.

The repository includes different mechanical parts designed to support the automation, handling, positioning, and adaptation of laboratory elements around the waterbath setup. These components were developed as practical design solutions for experimental workflows involving robotic manipulation and laboratory automation.

## Files Included

* FreeCAD files
* STEP files
* Waterbath-related mechanical components
* Falcon tube accessories
* Test tube trays and holders
* pH probe supports
* Balance and sample positioning fixtures
* TPU soft gripping components
* Additional laboratory automation parts

## Design Purpose

The components included in this repository were designed to improve the usability, modularity, and adaptability of the Waterbath experimental setup.

The designs are intended to be manufactured using 3D printing and can be modified depending on the specific requirements of each experimental configuration.
